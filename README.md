## Description

[Nest](https://github.com/nestjs/nest) microservice to process notifications.

## Installation

Insert the Upstash credentials in kafka-consumer.service.ts

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
